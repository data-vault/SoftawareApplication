Software Application extension list

This page defines a set of extension values for use with schema.org.

Each of these can be used as a value of the applicationCategory property, or as a type.

More Information

Schema.org can be extended in various ways. This is an example of an external enumeration extension, in which a list of terms is defined externally to the project. Other extension proposals lead to changes in schema.org itself, see the issue tracker for a list of proposals to change or extend schema.org vocabulary.

The terms defined here can be used in two ways:

  • As property values, e.g. for applicationCategory you might indicate that a software application was educational:

    <link itemprop="applicationCategory" href="https://data-vault.github.io/SoftwareApplication/#educationApplication"/>
  • As a type, e.g. via additionalType property in Microdata, via @typeof in RDFa, @type in JSON-LD.
  • Some applications may understand the simple textual codes ("EntertainmentApplication" etc.) in place of their full URL equivalents.
  • Refer to supporting applications to understand which usage is most appropriate.
  • See also structured data testing tool and W3C RDFa Distiller views of this extension.
